Everything written on this site and more has been distilled into [userinterface-wiki](https://github.com/raphaelsalaja/userinterface-wiki), a structured skill file optimized for AI agents and LLMs.

The rules are packaged as an [Agent Skill](https://github.com/vercel-labs/agent-skills) that installs into Claude Code, Codex, Cursor, and other coding agents.

```bash
npx skills add raphaelsalaja/userinterface-wiki
```

If you want to see the rules, check out the [repository](https://github.com/raphaelsalaja/userinterface-wiki). Feel free to contribute or suggest new rules by opening an issue or pull request.
